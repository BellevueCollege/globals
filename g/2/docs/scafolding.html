Global Settings

Requires HTML5 doctype

Globals makes use of certain HTML elements and CSS properties that require the use of the HTML5 doctype. Include it at the beginning of all your projects.

  1. <!DOCTYPE html>
  2. <html lang="en">
  3.   ...
  4. </html>

Requires Viewport to be set

Globals is a mobile-first responsive framework. You need to set the viewport on all projects inside the <head> tag.

  1. <meta name="viewport" content="width=device-width" />

IE support

To improve IE support, it's highly recommended that you use the IE=edge setting to force IE to render your website using the latest rendering engine the users' browser supports. Place this <meta> tag somewhere in the <head> tag

  1. <meta http-equiv="X-UA-Compatible" content="IE=edge" />
